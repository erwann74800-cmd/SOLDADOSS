# TSI/ESME AI in aviation group project (Module Code: C-250-01):

The subject of our project is to create an innovative system (in the aicraft domain) that uses AI in our code and thats answers to a current problematic that needs solving. Then we should have a working code and place it on github and explain our project.

____________________________________________________________________________________________________________________________________________________________________

# Our Topic:
SOLDADOSS: Survivor-Oriented Learning-Driven Drone System, our project is a drone that will search people after a distaster (naturral or human) that will use AI to avoid obstacles in order to find them the quickest and safest way.
